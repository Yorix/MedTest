Для правильной работы программы необходимо при запуске указать файл формата *.txt, отвечающий следующим требованиям.
1. В тексте вопросы расположены один за другим. Разделитель - перевод строки.
2. Перевод строки внутри одного вопроса и между вопросами допустим неограниченное чило раз.
2. Варианты ответа пишутся сразу после вопроса с новой строки каждый.
3. Варианты ответа номеруются буквами ЛАТИНСКОГО алфавита от "A." до "E." (обязательно с точкой, обязательно 5 штук, регистр значения не имеет).
4. Существует два способа указания верного ответа:
	а) Верный ответ пишется под пунктом "A."
	б) Верный ответ указывается методом добавления "+" перед буквой номерации без пробела. (Пример: "+b. вариант ответа")
5. Способы указания верного ответа нельзя смешивать (указание вариантов должно быть одного типа во всём текстовом файле).




Пример с первым ваниантом указания верного ответа:

Роженица 23 лет. При внутреннем акушерском исследовании: раскрытие шейки матки полное. Плодный пузырь отсутствует. Предлежит головка, в плоскости выхода из малого таза. Стреловидный шов в прямом размере выхода из таза, малый родничок ближе к лону. Каким размером головки родится плод при данном варианте предлежания? 
A. Малым косым 
B. Прямым 
C. Поперечным 
D. Средним косым 
E. Большим косым 
Данные внутреннего акушерского исследования родильницы через 6 часов после начала родовой деятельности: открытие маточного зева 5 см, предлежит головка плода, прижата ко входу в малый таз. Стреловидный шов в поперечном размере плоскости входа в малый таз, малый родничок слева. 
	О каком моменте биомеханизма родов идет речь? 
a. Сгибание головки. 
b. Разгибание головки. 
c. Внутренний поворот головки. 
D. Дополнительное сгибание головки. 
E. Внутренний поворот плечиков.
